Politique de confidentialité

Dernière mise à jour : 1er mars 2026

1. Introduction

Bostaz ("nous", "notre", "nos") exploite la plateforme d'automatisation de réservations via Instagram DM. Cette politique explique comment nous collectons, utilisons et protégeons vos données personnelles.

2. Données collectées

Nous collectons les données suivantes :

  • Informations de compte : nom du salon, adresse courriel, informations de facturation
  • Données Instagram : identifiants de messages entrants (sender_id), contenu des messages DM échangés via notre bot
  • Données Google Calendar : jetons d'accès OAuth, identifiants de calendrier, créneaux de disponibilité
  • Données de réservation : date/heure des rendez-vous confirmés, identifiant du client

3. Utilisation des données

Nous utilisons vos données pour :

  • Fournir le service d'automatisation des réservations via Instagram DM
  • Synchroniser votre calendrier Google pour afficher les disponibilités réelles
  • Traiter les paiements via Stripe
  • Vous envoyer des notifications de service
  • Améliorer notre plateforme

4. Partage des données

Nous ne vendons jamais vos données. Nous partageons uniquement les informations nécessaires avec :

  • Stripe : traitement des paiements
  • Meta (Facebook/Instagram) : envoi/réception de messages DM via l'API officielle
  • Google : lecture/écriture de votre calendrier via OAuth

5. Conservation des données

Vos données sont conservées tant que votre compte est actif. En cas de résiliation, vos données sont supprimées dans un délai de 30 jours suivant votre demande.

6. Sécurité

Vos données sont hébergées sur des serveurs sécurisés au Canada. Nous utilisons le chiffrement SSL/TLS pour toutes les communications. Les jetons d'accès sont stockés de manière chiffrée.

7. Vos droits

Conformément à la LPRPDE (loi canadienne sur la protection des renseignements personnels), vous avez le droit de :

  • Accéder à vos données personnelles
  • Corriger des informations inexactes
  • Demander la suppression de vos données
  • Retirer votre consentement en tout temps

8. Données des utilisateurs Instagram

Notre application accède aux messages Instagram uniquement dans le cadre de la fourniture du service de réservation automatique. Les messages sont traités automatiquement et ne sont pas consultés par des employés humains, sauf en cas de support technique avec votre consentement explicite.

9. Contact

Pour toute question concernant vos données personnelles : privacy@bostaz.com
